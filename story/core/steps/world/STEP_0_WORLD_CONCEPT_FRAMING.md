# STEP 0 — World Concept Framing

## Purpose

Define the world's operating premise in a form that later world-creation steps can use directly.

## Read inputs from

- The user's world brief
- Any inspiration anchor named by the user

## Save output to

- `<STORY_ROOT>/state/world_baselines/WORLD_CONCEPT.md`

## Skill use

- No skill required for this step.

## Task

Turn the brief into an original world concept that later steps can use for rules, geography, institutions, and daily-life work.

Write one concept that states:

- what kind of world this is
- how people normally live inside it
- what the world materially runs on
- what actions are ordinary
- what actions are exceptional or disruptive

## Required output blocks

### `WORLD PREMISE`

State the world's core idea in one concise paragraph.

### `WHAT PEOPLE DO EVERY DAY`

State the ordinary daily activities, obligations, or routines that define life in this world.

### `WHAT THIS WORLD RUNS ON`

State the material or social supports that keep daily life working in this world.

### `WHAT COUNTS AS NORMAL`

State what people in this world would treat as ordinary, expected, or widely accepted.

### `WHAT COUNTS AS EXCEPTIONAL`

State what people in this world would treat as unusual, disruptive, dangerous, or socially charged.

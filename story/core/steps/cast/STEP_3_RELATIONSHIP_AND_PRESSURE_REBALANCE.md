# STEP 3 — Relationship and Pressure Rebalance

## Purpose

Identify the specific relation links and pressure sources that need adjustment in the current cast.

## Read inputs from

- `<STORY_ROOT>/state/cast_management/CAST_SCOPE.md`
- `<STORY_ROOT>/state/cast_management/CAST_AUDIT.md`
- `<STORY_ROOT>/state/cast_management/CAST_MISSING_AND_OVERLAP.md`
- packaged characters under `<STORY_ROOT>/state/characters/`, when they exist

## Save output to

- `<STORY_ROOT>/state/cast_management/CAST_REBALANCE.md`

## Skill use

- No skill required for this step.

## Task

Turn the saved cast diagnosis into a local rebalance pass for the current cast set.

Write the result so a fresh sub-agent can see:

- which relation links need strengthening, balancing, or completion
- which pressure sources are concentrated, sparse, or missing
- which characters need more social anchoring or more practical pressure
- which concrete rebalance moves should come next for the cast

## Required output blocks

### `RELATION GAPS`

List the most important missing, one-sided, weak, or underdeveloped relation links.

### `PRESSURE GAPS`

List where pressure is missing, over-clustered, or unevenly distributed across the cast.

### `REBALANCE NOTES`

List the concrete rebalance moves that should be applied to improve cast structure.

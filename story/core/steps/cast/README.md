# Cast Creation Step Output Map

Use this file as the default output map for `CAST_CREATION_WORKFLOW.md`.

## Cast Management Output Area

Write cast-level working artifacts to:

- `<STORY_ROOT>/state/cast_management/`

Use these files for the main cast workflow outputs:

- `CAST_CHARACTER_LIST.md`
- `CAST_SCOPE.md`
- `CAST_AUDIT.md`
- `CAST_MISSING_AND_OVERLAP.md`
- `CAST_REBALANCE.md`
- `CAST_ACTION_REQUESTS.md`
- `CAST_QA.md`

## Character Creation Progress

Write the cast-to-character handoff progress to:

- `<STORY_ROOT>/state/cast_management/CHARACTER_CREATION_PROGRESS.md`

Keep the readable cast handoff list in:

- `<STORY_ROOT>/state/cast_management/CAST_CHARACTER_LIST.md`

## Acceptance Reporting

For each `STEP n / STEP n.5` block, report:

- files created or updated
- `PASS` or `FAIL`
- the blocker that stops the workflow before the next step

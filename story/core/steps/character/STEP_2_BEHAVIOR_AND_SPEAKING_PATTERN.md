# STEP 2 — Observable Behavior and Speaking Pattern

## Purpose

Convert the character concept into a usable behavior-and-voice contract for later writing steps.

## Read inputs from

- `<STORY_ROOT>/state/character_baselines/CHARACTER_CONCEPT.md`
- `<STORY_ROOT>/state/character_baselines/CHARACTER_WORLD_ROLE.md`

## Save output to

- `<STORY_ROOT>/state/character_baselines/CHARACTER_BEHAVIOR_AND_VOICE.md`

## Skill use

- No skill required for this step.

## Task

Read the concept and world-role notes, then write a scene-ready baseline that a later writer can apply without reinterpretation.

Make the output concrete and observable. Use specific habits, response patterns, and speech traits that could appear in dialogue or staging.

Do not repeat broad character themes unless you turn them into a visible behavior. Do not write prose, lore recap, or analysis. Do not add sections beyond the required blocks.

The finished file must answer these four questions:

- What does this character repeatedly do that can be seen on the page?
- What changes in their behavior when pressure rises?
- How do they sound in ordinary conversation?
- What do they consistently avoid saying or doing?

## Required output blocks

### `OBSERVABLE HABITS`

Write 3-6 short, concrete habits, routines, gestures, or visible choices.

Requirements:

- focus on actions other characters could notice
- include repeated or signature behaviors, not one-time events
- prefer specific verbs over abstract traits
- if useful, note when the habit appears

### `PRESSURE REACTION`

Write how the character behaves when interrupted, challenged, watched, rushed, blamed, cornered, or otherwise put under pressure.

Requirements:

- describe the default pressure response, not an extreme exception
- include body language, pacing, deflection, aggression, silence, or speed of response as relevant
- show whether pressure makes the character sharper, quieter, more formal, more evasive, more emotional, or more direct

### `SPEAKING PATTERN`

Write the character's normal speaking style in compact, practical terms.

Requirements:

- describe rhythm, sentence length, directness, formality, hedging, slang, repetition, or verbal habits
- note whether the character asks, states, jokes, avoids, flatters, or challenges by default
- if useful, include a short example pattern in plain description, not quoted dialogue

### `AVOIDANCES`

Write the behaviors and speech moves the character tends to avoid in everyday interaction and under stress.

Requirements:

- state what they will not readily say, admit, promise, reveal, or do
- include social, emotional, or physical avoidances only if they are part of the character's consistent pattern
- keep this block specific enough that later writing can preserve the same restraint
